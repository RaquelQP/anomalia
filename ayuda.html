¿Cómo funciona Anomalia?

Anomalia te ayuda a identificar enlaces sospechosos en tu correo electrónico. Aquí puedes aprender qué significa cada advertencia y cómo actuar.

Acortadores Enlaces a IPs Unicode Camuflaje Credenciales Redirecciones Homoglifos

Acortadores de URL

Los acortadores como bit.ly, t.co o goo.gl ocultan el destino real del enlace.

Ejemplo: https://bit.ly/3xYzAbC

Recomendación: Si no confías en el remitente, no hagas clic. Puedes comprobar el destino en ExpandURL.

Enlaces a direcciones IP

Un enlace que apunta directamente a una IP puede ser sospechoso.

Ejemplo: http://192.168.1.1/login

Comprueba la IP en Whois o VirusTotal.

Caracteres Unicode sospechosos

Algunos enlaces usan letras de otros alfabetos para parecer legítimos.

Ejemplo: paypaⅼ.com (la "l" es una letra griega, no una "l" latina)

Si ves un enlace raro, desconfía y revisa la dirección cuidadosamente.

Camuflaje tipográfico

Algunos enlaces usan caracteres similares para engañar (por ejemplo, "0" y "O", "1" y "l").

Ejemplo: micros0ft.com (con un cero en vez de "o")

Fíjate bien en los caracteres antes de hacer clic.

Credenciales en URLs

Si una URL contiene usuario y contraseña, puede ser un intento de engaño.

Ejemplo: http://usuario:contraseña@ejemplo.com

No introduzcas tus datos en sitios sospechosos.

Redirecciones sospechosas

Algunos enlaces usan parámetros como redirect, url o next para enviarte a otro sitio.

Ejemplo: https://ejemplo.com?redirect=http://malicioso.com

Puedes analizar el enlace en VirusTotal.

Homoglifos y caracteres invisibles

Algunos enlaces usan caracteres invisibles o de control para camuflarse.

Ejemplo: gоogle.com (la "o" es cirílica, no latina)

Si tienes dudas, copia el enlace y pégalo en un editor de texto para revisarlo.
